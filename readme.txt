Allows you to easily embed a SupporterWall into your site.

== Description ==

SupporterWall.com is a fundraising platform that is designed to be placed on your website. This plugin makes it easy to add a SupporterWall by providing a WordPress shortcode. It also properly differentiates between SSL and non-SSL versions of the script.

== Installation ==
1. Upload "supporterwall-shortcode.php" to the "/wp-content/plugins/" directory.
2. Activate the SupporterWall Shortcode plugin in the Plugins area of the admin panel.
2. Place [supporterwall key="YOUR_KEY"] in your templates where you want the wall to show up.
